# Twilio-Nodejs-SMS

[![Greenkeeper badge](https://badges.greenkeeper.io/pitops/twilio-nodejs-sms.svg)](https://greenkeeper.io/)

### Setup

First you need to have NodeJS v8+ installed and PM2 installed globally by running `npm install -g pm2`

and then run
```npm run setup```

### Run the app
```npm start```

### To test
```npm test```
